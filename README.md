# アプリケーション概要
地球温暖化を防ぐための小さなアクションを促し、毎日継続できるように、ユーザー同士で楽しんで取り組むことができます。

# 利用方法
### アクションを記録
1. ヘッダーからユーザー新規登録を行います
2. 「アクションをきろくする」リンクから、今日やったアクションを記録することができます

### ともだち同士でアクションを確認
1. ヘッダーの「マイページ」リンクから、ともだちがどんなアクションをどのくらい行なっているかを確認することができます。

# アプリケーションを作成した背景
社会的な課題であるSDGs、その中でも学生時代から興味のあった環境問題をテーマにしました。  
環境問題は、ESD(Education for Sustainable Development)「持続可能な開発のための教育」が重要であるといわれているため、毎日の暮らしの中で考えて学び気づかせる習慣づけが大切なのではないかと考えました。  
そのため、主に小学生を対象とした小さなアクションを積み重ねを促すようなアプリケーションを開発することにしました。

## ユーザー管理機能

**◆ 新規登録**  
ヘッダーのメニューから新規登録画面へ遷移し、  
「なまえ」「メールアドレス」「パスワード」で新規登録をします。  
アイコン画像は必須ではありません。

**◆ ログイン**  
ヘッダーのメニューからログイン画面へ遷移し、  
「メールアドレス」「パスワード」でログインします。  

**◆ ログアウト**  
ヘッダーのメニューからログアウトできます。  

**◆ ユーザー情報編集**  
ログイン後、マイページから「アカウント設定」ができるようになります。

**◆ ユーザーの検索**

## ともだち機能（相互フォロー機能）
### ともだちリクエスト
**◆ ともだちリクエストを送る**  
1. マイページのともだちリストから「ユーザー検索」をして、ともだちリクエストをしたいユーザーを見つけます。
2. 該当ユーザーのマイページに移動し、「ともだちリクエストを送る」ボタンを押します。  
ボタンは「ともだちリクエスト中」という文字に変わります。

[![Image from Gyazo](https://i.gyazo.com/62ab05a7c4f58fccbdfe5fd25a7818a2.gif)](https://gyazo.com/62ab05a7c4f58fccbdfe5fd25a7818a2)

**◆ お互いにリクエストをして「ともだち」になる**  
1. ともだちリクエストをされているユーザーのマイページに移動します。
2. 「ともだちリクエストを送る」ボタンを押すと、「ともだち」という文字に変わります。  
マイページに「ともだち」になったユーザーのアイコンが表示されるようになりました。

[![Image from Gyazo](https://i.gyazo.com/84a030b88536ed7aded42ca20af5d7bd.gif)](https://gyazo.com/84a030b88536ed7aded42ca20af5d7bd)

### ともだちのアクション実績を確認
２つの方法でともだちのアクション実績を確認することができます。  

**◆ マイページ**  
マイページにともだちの内３人が表示されます。  
アイコンの下の数字が「すべてのアクションの数」を表しています。

ともだちがどのくらいアクションをしているか確認することで、競争意識が働き、感化されることを期待しました。

## アクションリクエスト機能
アクションはカテゴリ別に３６個用意しています。  
これら以外のアクションをユーザー自らが自由に登録することはできません。  
そのため、アクションをリクエストするフォームを用意しました。

[![Image from Gyazo](https://i.gyazo.com/772f97a8409bc26b7a756f3c794606d2.gif)](https://gyazo.com/772f97a8409bc26b7a756f3c794606d2)

リクエストを送信すると、リクエスト内容が送信されたことを通知するために、ユーザーのメールアドレス宛に送信内容が自動で送られます。  
管理者宛にもメールが届くので、リクエスト内容を確認し、必要に応じてアクションを追加することができます。

# 工夫したポイント
**シンプルな操作**  
基本的にログイン以外の文字入力をユーザーにさせないようにしました。  
ねらいは以下の２点です。  
1. 『アクションを毎日継続するお手伝い』がコンセプトのため、煩雑な操作がないようにしました。
2. 小学生を対象としているため、ゲーム感覚で直感的に使えるようにしました。

**今日のアクション**  
ログイン後、トップページに「今日のアクション」をランダムで３つ表示するようにしました。  
どんなアクションをすればよいか分からない、あるいは、いつもと違うアクションに挑戦したいと思っている人のヒントになれば、という思いで実装しました。
